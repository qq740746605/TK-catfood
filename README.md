一、方案标题
宠物喂食器

二、方案应用场景
地点：家庭或宠物店使用，打工人可远程喂猫 猫猫少食多餐 自动喂猫，宠物店自动投喂节约人力；
功能：
1、手动按键投放固定量粮食；
2、定时、定量自动投放粮食（投食计划）；
3、APP远程控制：手动投放粮食，自定义投放量；
4、云端记录每天的投食次数；
5、APP设置投食计划；
6、投食器有蜂鸣器提醒宠物已投食；
